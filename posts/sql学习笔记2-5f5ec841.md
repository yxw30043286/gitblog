1.  求至少选修了学号为101的学生所选修的全部课程的学生学号和姓名。

```sql
select sno,sname
from student
where sno in(
	select sno
	from student_course sc1
	where not exists(
		select *
		from student_course sc2
		where sc2.sno='101' and not exists(
			select *
			from student_course sc3
			where sc3.sno=sc1.sno and
			sc3.tcid=sc2.tcid
		)
	)
)
```
