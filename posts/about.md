## 你好，这里是小红鸡

这是一个由我自己折腾、自己写、自己维护的小博客。如果你是从某篇技术文章顺着面包屑摸过来的，或者从老博客（[博客园](https://www.cnblogs.com/gitpull) / [flymysql.github.io](https://flymysql.github.io/archives/)）一路追过来，那挺好——这篇就是写给你的。

---

## 现在在做什么

我是一名已经工作的程序员，目前主要在做 **系统软件** 与 **数据库系统**——更具体一点，是分布式数据库 / 图数据库 / 查询引擎一类的东西。日常打交道的事情大致这几样：

- 查询的解析、计划生成、执行算子；
- 存储层、索引结构、内存与磁盘上的数据布局；
- 分布式系统里那些老问题：分片、调度、副本、一致性；
- 偶尔写点服务端工具，给上面这些事打杂。

总体上，我喜欢那种"看得见执行路径"的代码——一条 SQL / 一段图查询语言扔进去，能从语法树一路追到磁盘 IO 的那种。这个博客里很多技术文章，都是在把这条路径上的某一段拆开来给自己讲一遍。

## 这里在写什么

我把这个博客当作自己的 **工程师手账**：技术、生活、和介于两者之间的东西。

技术方面，大致可以归到这几条主线（点标签可以看全部）：

- [图数据库 / 图计算](tags.html#图数据库)：从 Nebula Graph 的源码走读，到日常对查询执行的一些理解；
- [数据结构](tags.html#数据结构) / [算法](tags.html#算法)：从 PAT 乙级到 Dijkstra 单源最短路径，再到工程里一些更朴素的小算法；
- [推荐系统](tags.html#推荐) / [推荐算法](tags.html#推荐算法)：写过推荐引擎架构的概览，记一些常见的工程思路；
- [前端](tags.html#前端) / [微信小程序](tags.html#微信小程序)：早年做过几个微信小程序（"小鸡背单词"系列）、撸过 vue.js 的小组件、改过自己博客的样式；
- [博客建站](tags.html#博客建站)：博客从 Hexo → WordPress → CDN → 现在的 GitHub Pages 伪后台，一路上踩过的坑都还摆在那里。

生活方面，主要是一些[随想](tags.html#随想)。早年还写过一段公众号文字（署名 [邻家酒肆](tags.html#邻家酒肆)）——一些不那么程序员的东西，关于成长、关于喜欢、关于离开和回来的故事。如果你顺着[归档](archives.html)翻下去看到一篇画风突变的，那大概就是了。

## 一点点小历史

我并不是天生很强的人。现在写得出来的东西，几乎都是当年作为一个普通本科生，从一道道 PAT 的 C++ 题、一段段 SQL 练习、一个个 Hexo / WordPress 折腾里一点点攒出来的。这个博客里的老文章，多半就是那时候记的笔记。

后来毕业了、参加了工作、换了几次方向，开始啃工业级系统的代码——也才慢慢意识到：很多以前以为离自己很远的东西，其实都是从最朴素的几样基本功长出来的。所以这个博客也保留着早年那些小白笔记，没有删——它们多半粗糙，但没什么不能拿出来的。

## 关于这个博客本身

这个博客本身也是一个折腾产物：完全静态托管在 **GitHub Pages**，但带一个 **纯前端的"伪后台"**——可以直接在浏览器里写文章、上传图片、改设置、调主题，再通过 GitHub Contents API 把改动一笔一笔推回仓库。从这个意义上讲，每一次"发文章"的本质都是给自己的仓库提一个 commit。

如果你也在折腾 GitHub Pages 博客，可以翻翻 [博客建站](tags.html#博客建站) 标签下的那些文章，多半能找到一两个你也踩过的坑。
